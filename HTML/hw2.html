DEFINITION OF INHERITANCE:- One class properties inherits from another class with the keyword EXTENDS

TYPES OF INHERITANCE

1. Single level inheritance:- once single class extends another class.
2. Multi level inheritance:- child class inherited by parent class, and that child class inherits another sub class.
3.Hierarchical inheritance:- one parent class inherits multiple child class.
4. Hybrid inheritance:- combination of two or more inheritance.
5. Multiple inheritance:- child class inherits from 2 parent class, it is not used in inheritance but in Interface concept.

Loops in java

DEFINITION OF LOOPS:- Loops are used when we want to repeat some code again and again until a condition is true.

TYPES OF LOOPS IN JAVA

1. For Loop:- We use it when we know how many times we want to repeat the code.
2. While Loop:- We use it when we dont know the number of times, but we keep repeating as long as the condition is true.
3. Do-While Loop:- This loop runs the code at least one time, even if the condition is false, because the condition is checked after running the code.
4. For-Each Loop:- This loop is mainly used for arrays and collections. It is the easiest way to go through each item one by one.

DEFINITION OF ENCAPSULATION:- Encapsulation means wrapping (hiding) the data and methods together in a single unit (class).

KEY POINTS OF ENCAPSULATION

# Data Hiding:- The variables of a class are kept private so that they cannot be accessed directly from outside. # Use of Getters and Setters:- We use getter methods to read the data and setter methods to change the data in a safe way. # Security:- It protects data from being changed accidentally or wrongly by other classes. # Flexibility:- We can control how the data is accessed or updateds , making the code easy to maintain.

MODIFIERS IN JAVA

Access Modifiers:-
private = not accessible outside the class.
protected = accessible in inheritance (child class), even outside package.
default = accessible only within the same package, not outside package.
public = accessible everywhere in the project.

Non-Access Modifiers:-
static = belongs to the class, not objects.
non-static = belongs to the object, each object has its own copy.

POLYMORPHISM IN JAVA

Polymorphism means one task can be performed in many ways.
There are two types of polymorphism in Java.
Compile time polymorphism is called method overloading. It happens when methods have the same name but different parameters or datatypes.
Runtime polymorphism is called method overriding. It happens when a child class provides a new implementation of a method that is already defined in the parent class.
Method overloading is resolved at compile time.
Method overriding is resolved at runtime.
